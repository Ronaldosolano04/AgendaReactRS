Este proyecto consiste en una Agenda Web desarrollada con la librería React que permite visualizar y agregar contactos utilizando una API externa. La aplicación está estructurada mediante componentes, donde el componente padre **App** se encarga de gestionar el estado general de la aplicación y de realizar las peticiones HTTP utilizando la función **fetch**. A través del método **GET** se obtienen los contactos almacenados en el servicio web `http://www.raydelto.org/agenda.php`, los cuales se muestran en pantalla mediante el componente **ListaContactos**, encargado de recorrer el arreglo de contactos y presentarlos en una tabla con su nombre, apellido y teléfono. Por otro lado, el componente **AgregarContacto** contiene un formulario que permite introducir los datos de un nuevo contacto y enviarlos al servidor mediante una petición **POST** en formato JSON. Una vez agregado el contacto, la aplicación vuelve a consultar la API para actualizar automáticamente el listado mostrado. De esta manera, el proyecto demuestra el uso de React con componentes, manejo de estado con **useState**, ejecución de efectos con **useEffect**, comunicación entre componentes mediante **props** y consumo de servicios web utilizando **fetch**, cumpliendo con los requerimientos de crear una agenda interactiva que permita listar y registrar contactos en línea.
